
QGAN Super Resolution Web Application

Overview
The "QGAN Super Resolution Web Application" is an AI-powered project that enhances low-resolution images into high-resolution images using advanced deep learning techniques and Quantum Generative Adversarial Networks (QGAN).

This project demonstrates the integration of "Artificial Intelligence + Web Development" to solve real-world image enhancement problems.

Objective
To design and develop a web-based system that:
Takes low-quality images as input
Enhances image resolution using AI-based models
Provides high-quality reconstructed images
Supports medical, satellite, and general image enhancement use cases

Key Features-
Upload low-resolution images
AI-based super-resolution processing
High-quality image output generation
User authentication system (Login/Register)
Database integration for storing results
Web-based interface using Flask
Fast image processing pipeline
